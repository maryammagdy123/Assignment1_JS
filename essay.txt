
** What is the difference between forEach and for...of? When would you use each?
forEach:
Runs a function on every element in an array.
You cannot use break or continue.
Use it when you just want to loop through all elements without stopping.

for...of:
Works with any iterable (like arrays and strings).
You can use break and continue.
Use it when you need more control over the loop.

2. What is hoisting and what is the Temporal Dead Zone (TDZ)? Explain with examples.

Hoisting: declarations are moved to the top; var becomes undefined, functions are usable before their declaration.

TDZ: accessing let or const before their declaration causes an error.


hoisting ->
let x=5;
let y=5;
sum(x,y)
function sum(x,y){
	console.log(x+y)
	return x+y;
} 
// function can be invoked before the bode function

TDZ->
sum(5, 5); 
// Error: Cannot access 'sum' before initialization

let sum = (x, y) => {
  console.log(x + y);
  return x + y;
};
// let and const cannot be used before their declaration

4.Explain how try-catch works and why it is important in async operations.

Try-catch runs code in the try block and catches errors in the catch block to handle them without stopping the program. It is important in async operations to catch errors in async/await or API calls and prevent crashes.
